## xxd使用

https://www.xiebruce.top/1789.html

查看二进制
```
xxd -b
```

设置显示字节数, 例如显示0x30个字节数
```
xdd -l 0x30 /path/to/test.txt
```

设置每行显示字节数
```
xxd -c 6 /path/to/test.txt
```

跳过n行
```
xdd -s 0x30 /path/to/test.txt
```

连续输出十六进制
```
xxd -ps /path/to/test.txt
```
